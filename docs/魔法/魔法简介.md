# 总览

魔法是每一个小魔女的必学本领，也是魔女高考重中之重的科目

## 考点：魔力

魔力最常见的制取方法是通过食物获取

你可以通过 [献祭法阵](../法阵/献祭法阵.md) 来批量制取魔力

通过 [血液连接](../法阵/法阵简介.md) 来导通魔力

从本质而言，魔女的胃也是一个献祭法阵，用来提供源源不断的魔力，也可以向外向法阵等建筑释放魔力

魔力的基本单位是 “诺亚”

通过测量可知，存在魔力守恒

## 考点：法阵

[法阵](../法阵/法阵简介.md) 的使用也是必修课程

## 考点：烹饪

[烹饪](./烹饪.md) 的基础使用也是必修课程，对应更高阶的 [炼金](./炼金.md) 则为选择性必修
