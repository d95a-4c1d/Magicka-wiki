# 剧情

由于平行宇宙的相似性，人类文明在各种平行宇宙中的发展进程都大差不差，发展出了现代科学体系。而有一个平行宇宙的参数稍微有点不一样，那里没有正常的人类文明，只有“小魔女”们在那里平淡而幸福地生活着。

根据小魔女的传说，很久很久以前，这个平行宇宙里只有大魔女孤零零一个人存在着。没有人记得她的长相、性格、身世，只知道她叫诺亚。有一天，诺亚感到十分无聊，就以自己为模板，利用复制法阵的升级版——克隆法阵创造了一群小人（相对于诺亚的体型），也就是后来的小魔女。诺亚运用法术非常熟练，随手就画出了克隆法阵。但她画到一半时手不小心抖了一下，克隆法阵就有了瑕疵，导致克隆出来的小魔女有高矮胖瘦，都长得和她不太一样。小魔女们都继承了诺亚的魔力和魔法知识，也能够熟练地画出各种法阵，但似乎都有点憨憨的，没法领会法阵背后的原理，也就没法帮助推进对魔法或者宇宙规律的研究。对此，诺亚在晚年感到很苦恼，郁郁而终。唯一让诺亚宽慰的是，即使要离开她的智慧，小魔女们也没有放弃研究魔法的进程。她们不知疲倦地倒腾各种法阵的组合，试图凭借运气或者直觉来试验出更强大、功能更丰富的法阵。在常年累月的研究下，各种实验记录已经堆积成山，甚至塞满了世界各地的图书馆。

其实小魔女们也不很理解研究魔法的意义是什么，只知道诺亚临死前给她们许下的心愿：合成出能将石头变成金子的点金石。但只要稍微了解一下魔法的研究历史，就知道这是几乎不可能的。因为以往所有的法术都只发生了宏观层面的变化，对于分子尺度以下的过程则没有丝毫办法。而将石头变成金子，本质上是将一种原子转化为了另一种原子，这在宏观的法术体系下是不存在的，凭借小魔女们的小脑瓜，也很难对法术体系做出突破。因此，在大魔女死后的几百年，小魔女们几乎没有实质上的进展。

直到有一天，小魔女们将传送法阵魔改到了极致，成为了位面跃迁法阵。她们在一个没有大气的固体行星表面作画，因为这样可以才可以画出足够大的法阵，而且能保存很长时间。在把星球表面密密麻麻画满奇形怪状的纹路以后，她们激活了这个法阵，将几名探险队员传送到了其中一个普通的平行宇宙里。她们抵达时，发现人类文明早已在和AI的战争中同归于尽，只留下残垣断壁和对昔日辉煌的只言片语，以及由于中心服务器被人类猛烈的反抗击毁而四处游荡的失控机器人。她们四处搜集前文明的资料，并发现他们竟然对核物理有着深入研究，甚至已经制造出可以产生新原子的机器——粒子对撞机！探险队员们喜出望外，把这个好消息带回到了魔女宇宙。从此以后，越来越多的探险队启程前往这一个世界，并尝试着带回前文明的足够资料，从而复原出粒子对撞机，实现点石成金的梦想。
