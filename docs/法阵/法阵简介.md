# 法阵

法阵是使用魔法的基础设施。

一般而言，法阵通过粉末绘制而成，绘制完成后，将鼠标放置于上方，按下 “E” 键以启用法阵，此时粉末会溶解，并在地板上留下印记

一部分法阵需要 “魔力输入” 或 “原料输入”

长按 “B” 键以使用血液作画，血液可以联通法阵(可以理解为电线与传送带的结合)

## 技术细节
按下 “E” 键后，向朝向方向发射射线，以第一个碰撞到的粉末线条为基础，求解出线条的联通块

紧接着，用一个最小的 **正方形** 包围盒将其包围，随后将线条栅格化至 256X256 的图像，在此基础上跑CNN求解出法阵类型
