# Magicka

Magicka 是这个项目的暂定名称。

我们设想一款以简笔画和煲汤驱动的法术战斗旅行游戏。你可以在地上画出各种魔法阵，以此来制造、传送、塔防甚至复活队友等，同时通过烹饪（煲汤）得到各种神奇的食物和材料，最后抵达旅途的终点。

此项目虽然目前是公开项目，但也请尽量避免不必要的曝光。

## 文档编写教程

### 准备工作
1. 用邮箱注册一个 github 账号；
2. 如果要成为贡献者 (以便自由更改文档)，联系仓库管理员 (d95a-4c1d)，把账号添加进贡献者名单。
   
### 对于不熟悉 git 工作流的贡献者

#### 修改现有页面
1. 点击你想修改的页面的右上角的书和笔图标进入编辑页面；
2. 编辑完毕以后点击编辑页面右上角的绿色 `Commit changes` 按钮；
3. 在弹窗的 "Commit message" 那一栏里写这次编辑的说明，然后点击下方绿色 `Commit changes` 按钮。
   
#### 添加新页面
1. 访问 [https://github.com/d95a-4c1d/Magicka-wiki/new/main/docs](https://github.com/d95a-4c1d/Magicka-wiki/new/main/docs)；
2. 点击右上角 `Add file` 下的 `Create new file`；
3. 先在上方输入框输入文件名, 然后在下面编辑内容；
4. 接下来是和修改一样的流程。

### 对于熟悉 git 工作流的贡献者
1. 阅读 MkDocs 的教程，安装 MkDocs ；
2. 用教程中演示的方式修改并测试网站内容；
3. 确认无误后 commit 到 main 分支；
4. 用命令 `mkdocs gh-deploy` 将修改同步到 github pages 页面，目前的网址为 [https://d95a-4c1d.github.io/Magicka-wiki/](https://d95a-4c1d.github.io/Magicka-wiki/) (可能需要等会才更新内容)。

### 对于其他人
使用 issue 等方式联系仓库管理员。
